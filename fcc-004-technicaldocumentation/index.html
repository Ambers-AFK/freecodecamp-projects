Overview

What is Lua?

Lua is a lightweight, high-level, multi-paradigm programming language designed primarily for embedded use in applications. Lua is cross-platform, since the interpreter of compiled bytecode is written in ANSI C, and Lua has a relatively simple C API to embed it into applications.

Setup

Installation setup on Linux (Debian / Ubuntu based distro)

To get the newest version of Lua (Lua 5.4) just type into your terminal:

$ sudo apt-get install lua5.4

And you should be good to go.

To test to see if everything is working properly you can type into your terminal:

$ lua -v

The output of that command should be something similar to

Lua 5.4.2 Copyright (C) 1994-2020 Lua.org, PUC-Rio

It is pretty easy to get started with Lua on other Linux distros, Windows and MacOS, but the installation process for them will not be covered here.

Getting started

Now that you have Lua installed, how do you use it?

There are two main ways of using Lua, those being through the interactive shell and using lua to launch a program from a file.

The interactive shell is the program that gets called when you simply type Lua in the terminal without any other arguments it looks something like this:

$ lua Lua 5.4.2 Copyright (C) 1994-2020 Lua.org, PUC-Rio >

You can go ahead and write your programs in here, but it is not going to be very efficient and you won't really be able to save your work without some headaches. The interactive shell is great if you want to test a feature you are working on.

The ogligatory "Hello, World!" program will look something like this in the interactive shell:

$ lua Lua 5.4.2 Copyright (C) 1994-2020 Lua.org, PUC-Rio > print("Hello, World!") Hello, World! >

But if you want to write code, you probably will want to do so in an external file, for that you simply type inside the terminal for example for a file called main.lua:

$ lua main.lua Hello, World!
Comments

Comments are a very important part of every programming language, these can be notes written by programmers for themselves or other programmers, or any other kind of message that you don't want the computer to interpret.

In Lua to write a single-line comment you type -- followed by the comment

--this is a single-line comment

To write a multi-line comment you type --[[ to begin and ]] to end the comment

--[[ This is a multi-line comment! ]]
Variables

Variables are a way to store information, these are usually compared to boxes, the values stored in these variables can be of a couple different data types such as:

  • nil
  • boolean
  • number
  • string
  • function
  • userdata
  • thread
  • table

There are global and local variables, if you don't know what you're doing it is best practice to use for the most part only local variables.

You assign a variable like this:

either

local var_name = data

or

var_name = data

Where var_name is the name of your variable and data is the data you want to store

You can also add a new special type of variable called a constant, which as the name implies cannot be changed, you can do that by typing local var_name <const> = data but keep in mind that this was added in the newer versions of Lua only, so if you work with older versions of it, this will not work.

Operators

An operator is a symbol that indicates Lua that there is a specific manipulation to be done with some inputs.

There are a number of operators, including:

Matematical operators:

  • + addition
  • - substraction
  • * multiplication
  • / division
  • ^ exponentiation
  • % modulus

Relational operators:

  • == equal
  • ~= not equal
  • > greater than
  • < less than
  • >= greater than or equal
  • <= less than or equal

Logical operators:

  • and
  • or
  • not

Misc. operators:

  • = assignment
  • .. concatenation
  • # length return
Conditionals

Conditionals are as the name implies things that based on set conditions do something.

There are 3 words that can be used to define a condition:

  • if
  • else
  • elseif

After if you need to put in brackets a boolean operation that can be either true or false to be verified.

After if and the condition to be verified you also need to write the keyword "then".

If you want to have a condition for when the condition you are verifying is false you can do that with the "else" keyword.

If you want to verify different criteria you can also use the "elseif" keyword, but you need to do so before using the "else" keyword.

At the end of the conditional you need to use the "end" keyword.

A conditional statement will look something like this:

if (boolean) then do something elseif (boolean) do something else else do something end
Functions

In computer programming there is the concept of keeping your code "DRY" (Don't Repeat Yourself). And functions help us achieve just that. These are reusable pieces of code that we can use when we don't want to repeat ourselves.

There are different ways of making a function:

Like this:

function name_function() something to be done end

Like this:

function() something to be done end

Or like this:

var_name = function() something to be done end

In between the paranthesis you can add arguments to be used in your function, for example if you want to make a function that adds two numbers you could write it as function(x, y), where x is one number and y is the other number.

Loops

Loops are constructs used for when you want to repeat something until a condition is no longer true.

There are three types of loops in lua:

  • while loops
  • for loops
  • repeat until loops

The differences between them are, that while loops only check the condition, for loops check the condition and update the variable used until that condition is met, and repeat until loops are like while loops but are executed at least once, the condition being checked is only checked at the end.

A loop is similar to a conditional, but you need to write the "do" keyword instead of the "then" keyword. Except for the repeat until loops.

The loops will look something like this:

while(boolean) do something end for i = int, min/max, step do something end repeat something until (boolean)
Tables

Tables are a way of storing data. The most basic table can be made using:

local table_name = {table_entry1, table_entry2, (etc)}

You can access the data in a table using table_name[x], where x is a number.
